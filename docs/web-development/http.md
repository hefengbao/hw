# HTTP


- `Strict-Transport-Security`：强制 HTTPS 传输，预防中间人攻击
- `X-Frame-Options: DENY`：防止点击劫持攻击
- `X-XSS-Protection`：内置 XSS 防护机制
- `Content-Security-Policy`：严格的内容安全策略
- `Referrer-Policy`：控制引用信息泄露
