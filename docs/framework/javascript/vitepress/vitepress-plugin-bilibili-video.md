# Vitepress 引用哔哩哔哩视频

## 使用 `iframe` 实现

直接添加代码即可，注意修改 `bvid` 为要引用的视频 ID:

```html
<iframe 
	height=400px 
	width=100% 
	src="//player.bilibili.com/player.html?bvid=BV1vE421w7eu&page=1" 
	scrolling="no" 
	border="0" 
	frameborder="no" 
	framespacing="0" 
	allowfullscreen="true"> 
</iframe>
```
